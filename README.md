# Calculadora Flutter

Este projeto é uma calculadora simples desenvolvida com Flutter, com foco em praticar estruturação de UI, lógica de operações matemáticas e organização de widgets.

A aplicação permite realizar operações básicas de forma rápida e direta, com uma interface leve e responsiva.

## Funcionalidades

- Operações básicas: soma, subtração, multiplicação e divisão
- Interface simples e funcional
- Atualização do resultado em tempo real
- Estrutura separada em widgets e lógica

## Tecnologias utilizadas

- Flutter
- Dart

## Estrutura do projeto

O projeto está organizado da seguinte forma:

- `main.dart`: ponto de entrada da aplicação
- `pages/`: contém a tela principal da calculadora
- `widgets/`: componentes reutilizáveis da interface
- `utils/`: lógica das operações matemáticas

## Como executar

1. Clone o repositório
2. Execute `flutter pub get` para instalar as dependências
3. Rode o projeto com `flutter run`

## Objetivo

Este projeto foi criado com fins de estudo e prática, focando em melhorar habilidades com Flutter, organização de código e construção de interfaces.
